# Тестовое задание на backend-разработчика

Приложение библиотеки со списком книг и авторов.

[Ссылка на тестовое](https://docs.google.com/document/d/1T5vIh10-fKWAJmFJI2KUuj8KNv1yzKsTPRVAm0b7uEM/edit?tab=t.0)

### Основной функционал:

- Просмотр списка всех книг
- Просмотр списка всех авторов
- Добавление новых книг
- Добавление новых авторов
- Редактирование информации о книгах
- Редактирование информации об авторах
- Удаление книг и авторов

### Дополнительный функционал:

- Простая регистрация с ролями
- Middleware функционал
- Пагинация для таблицы

## Технологии

- **PHP** 8.4
- **MySQL**

## Запуск проекта в Docker

### Требования

- Установленный Docker
- Установленный Docker Compose

### Инструкция по запуску

1. **Клонируйте репозиторий**

```bash
git clone https://github.com/deniskeller/books-library-test-task
```

2. **Запустите контейнеры**

```bash
  docker-compose up -d
```

3. **Откройте приложение в браузере**

```bash
  http://localhost:8080/
```

### P.S.

В файлах .env.dev и config/config.php есть настройки для Open Server. Надо просто закоментировать docker-настройки на openserver-настройки. В варианте с запуском докер некорректно работают Middleware-ы. Попозже разберусь с этим.
