# Тестовое (книги на laravel)

Разработать сайт библиотеки со списком книг и списком авторов, используя php-фреймворк Laravel.

[](https://docs.google.com/document/d/1T5vIh10-fKWAJmFJI2KUuj8KNv1yzKsTPRVAm0b7uEM/edit?tab=t.0)

### Основной функционал:

- Просмотр списка всех книг
- Просмотр списка всех авторов
- Добавление новых книг
- Добавление новых авторов
- Редактирование информации о книгах
- Редактирование информации об авторах
- Удаление книг и авторов

## Технологии

- Чистый **PHP** 8.4
- **MySQL**
- **Composer** для управления зависимостями
